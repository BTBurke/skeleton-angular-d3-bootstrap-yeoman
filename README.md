skeleton-angular-d3-bootstrap-yeoman
====================================

Project skeleton using Angular JS, Bootstrap, D3, JQuery, Coffeescript and Yeoman for build environment

Components:

* AngularJS ~1.1.1
* Bootstrap ~2.2.0
* D3 ~3.0
* Font-Awesome ~3.0
* JQuery ~1.8.3

Uses Yeoman as a build environment, SCSS, Coffeescript
